Product narrative

Inference belongs in the IDE — not beside it.

RawrXD is a native Win32 IDE with first-class agent loops, speculative decoding, and a terminal model that matches VS Code / Cursor — without shipping Electron or a generic chat bolt-on.

Competitive framing

Ollama · vLLM · RawrXD

Three different layers — complementary, not interchangeable.

Ollama

Local model runtime & UX

Daemon pulls models, exposes a simple HTTP API, optimizes for “run Llama on my machine.” Great for scripts and apps — not an editing surface.

vLLM

Throughput-oriented serving

PagedAttention, continuous batching, multi-GPU — built for datacenter SLAs and OpenAI-compatible fleets. Operate inference as a service, not as an IDE feature.

RawrXD

IDE-native inference + agents

GGUF loaders, AVX-heavy paths, agent terminals, tool mirroring, and Win32 UI in one process. Positioning: where developers already work — with parity keyboard model and read-only agent surfaces like Cursor.
