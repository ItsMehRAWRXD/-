// PERFORMANCE CONSIDERATIONS
// ============================================================================
// 
// FlashAttention provides the most benefit when:
// 1. Sequence length is long (> 512 tokens)
// 2. Multiple heads are processed together
// 3. Memory bandwidth is the bottleneck
// 
// For short sequences or single-head attention, the overhead of
// reorganizing data for FlashAttention may outweigh the benefits.
// 
// The integration layer automatically falls back to standard attention
// when FlashAttention is not available or when it would be slower.
